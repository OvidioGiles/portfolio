PLC Remote Monitoring

Description

Worked with industrial technology company to monitor PLC electrical signals remotly and display results in dashboards for a quick revision of the system integrity. Project for internal company usage.

Company is dedicated to install water treatment plants. The objective of the project was to receive alerts when a sensor detects anomalies in the system.

Technology Involved

  • Siemens Logo PLC programing.
  • IMON RS485 Industrial transmitter programing.
  • GPRS and 3G signals.
  • HTML, CSS, Javascript (Chart.js), PHP and MySQL to manage information and create dashboards.

Company Name

Sistemas Eléctricos y Electrónicos Especializados S.A. de C.V.

Status of the project

Prototype implemented and documented for internal company usage.

Warehouse Optimization Project

Description

Worked as team member in a multidisciplinary team, to design a process and a software to store the products and take advantage of the available space due to special customer requirements.

The objective was to set a process for using extra space in a storage warehouse of the factory. The orders of the customer where more than the warehouse capacity and the available space was used to store finished goods and deliver the orders on time.

Technology Involved

  • Industrial process design and calculations to avoid wastes.
  • RFID tags and readers to track the extra finished goods and locations.
  • C# and MySQL software to display information to users (Warehouse supervisor and Production leaders).

Dynamic Locations Storage Software

Description

Worked as a team member developing a software to dynamically store work in progress material according to the customers usage and order specifications.

The objective was to redesign the standard process and avoid using space to store material with low usage frecuency. The project saved 220 square meters and started the process to become the new standard.

Technology Involved

  • Industrial process design and calculations to avoid wastes and implementation.
  • Solid Work rack design.
  • Barcode readers.
  • C# and SQL Server software to process information and asign dinamically the WIP to locations according to the company database verifying orders and configuration of the assembly.

Leoni Wiring Systems de Yucatán S.A. de C.V.

Status of the project
